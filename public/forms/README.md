# Application Forms Directory

This directory contains downloadable PDF forms for various embassy services.

## Required Forms

Please add the following PDF forms to this directory:

1. **visa-application.pdf** - Visa application form
2. **marriage-certificate-application.pdf** - Marriage certificate application form
3. **birth-certificate-application.pdf** - Birth certificate application form

## Instructions

- Each form should be a fillable PDF that users can download, complete manually, and submit by mail or in person
- Forms should match the official embassy documentation standards
- Keep filenames exactly as specified above for proper linking from the website
